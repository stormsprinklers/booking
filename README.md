# Storm Booking

## Setup

1. Install dependencies: `npm install`
2. Run locally: `npm run dev` (or your project's start command)
3. Build: `npm run build`

## Deploy to Vercel

Connect this repo to [Vercel](https://vercel.com) — it auto-detects most frameworks (Next.js, React, Vue, etc.). Push to GitHub and import the repository in the Vercel dashboard.

```bash
# Or deploy via CLI
npx vercel
```

## GitHub

Push to GitHub to sync and trigger Vercel deployments (if connected):

```bash
git remote add origin https://github.com/YOUR_USERNAME/Storm-Booking.git
git push -u origin main
```
